Charge It Privacy Policies

Charge It never collects any type of user data.

If you need more information about Charge It please contact us for Charge It at: chargeit.milkysoft@gmail.com

Charge It Permissions

1: Uses Permission: (Internet): This permission is required to keep you connected with the online world and to do some online activities.

2: Uses Permission: (Access Wi-Fi State): This permission is required to read Wi-Fi state.

3: Uses Permission: (Change Wi-Fi State): This permission is requires to change Wi-Fi state.

4: Uses Permission: (Bluetooth): This permission is required to read Bluetooth state.

5: Uses Permission: (Bluetooth Admin): This permission is required to change Bluetooth state.

6: Uses Permission: (Read Sync Settings): This permission is required to read Sync settings.

7: Uses Permission: (Write Sync Settings): This permission is required to change Sync settings.

8: Uses Permission: (NFC): This permission is required to control NFC.

9: Uses Permission: (Write Settings): This permission is required to change system settings to disable system features and functions.

10: Uses Permission: (Access Network State): This permission is required to read network state.
